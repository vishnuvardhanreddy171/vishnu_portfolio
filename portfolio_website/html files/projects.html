Acute Lymphoblastic Leukemia Classification using CNN

This project focuses on the automatic classification of Acute Lymphoblastic Leukemia (ALL) using deep learning techniques, specifically Convolutional Neural Networks (CNNs). The aim is to assist medical professionals in diagnosing ALL through image-based analysis of blood smear samples.

  • Preprocessing of blood cell images (resizing, normalization, noise removal)
  • Construction and training of a CNN model
  • Model evaluation using accuracy, precision, recall, and F1-score
  • Visualization of training and validation performance graphs
